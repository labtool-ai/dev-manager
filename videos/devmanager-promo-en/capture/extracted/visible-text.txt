DevManager — 本地 dev 进程管理器(by labtool)

一句话:一屏管好所有本地服务。

痛点:本地跑一堆 dev 服务(前端、后端、electron、多个端口),终端开一堆、来回切、忘了哪个在跑、端口撞了不知道谁占的。

核心功能:
- 项目侧栏 + 启动组合(Profiles):所有 dev 项目一个列表,按 tag 分组,一键批量启动一组。
- 一键 启停 / 重启:每个项目独立控制,支持自动重启。
- 彩色实时日志:ANSI 颜色透传,可搜索、可持久化。
- 实时指标:CPU / 内存 / 运行时长。
- 端口就绪自动识别:探测到端口 ready 自动可开浏览器;端口冲突能识别占用者(本 app 的项目 vs 外部进程),绝不盲杀。
- 局域网二维码:手机同一 Wi-Fi 扫码直接打开,调试移动端。
- ⌘K 快速启动面板 + ⌘⌥K 全局热键。
- 崩溃 / 就绪 通知。
- 菜单栏常驻:不开主窗也能快速启停。
- 使用统计 + GitHub 风格启动热力图。
- 深色 / 浅色 自动跟随系统。
- Sparkle 自动更新。
- MCP 接入:一行命令,让 Claude Code / Codex / Cursor 等 AI 工具直接 列 / 启 / 停 / 重启 / 新建 / 删除 项目、看日志、跑启动组合。

品牌:labtool · 主色 薄荷绿 #2DD4BF · 炭灰 #1B1B1F · 米白 #F5F5F4 · macOS 原生 SwiftUI 应用。

分镜草稿(备用解说词 / 屏幕字):
1. 痛点开场:本地服务开开关关,很烦?
2. 主界面:DevManager,一个侧栏全管起来。
3. 新建进程:填目录和命令,10 秒加一个项目。
4. 运行中(核心):启动、日志、指标、扫码调试,一屏搞定。
5. 统计热力图:你的本地开发,数据看得见。
6. MCP:一行命令,让 AI 直接帮你管进程。
7. 明暗转场:深色浅色,自动跟随系统。→ 落 logo + labtool
