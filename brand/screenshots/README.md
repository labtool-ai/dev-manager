# DevManager 产品截图说明

用于宣传图 / 视频制作的素材清单。全部为 macOS 原生窗口截图,Retina 2×,自带透明阴影(PNG alpha),可直接叠加到任意背景。

- **尺寸**:窗口图均 2130×1680;菜单栏图 628×442
- **配套主图**:`../hero.png`(浅色)· `../hero-dark.png`(深色),生成脚本 `../build_hero.py` / `../build_hero_dark.py`

## 01 · 主界面(空状态)
`devmanager-01-main-empty@2x.png` — 2130×1680

- **画面**:左侧栏(搜索框、PROFILES 启动组合、项目列表),右侧空白引导「选择一个项目」。窗口为激活态(红绿灯高亮)。
- **可见功能**:项目侧栏、启动组合(Profiles)、按 tag 分组、搜索。
- **解说 / 字幕**:「所有本地 dev 服务,一个侧栏管起来。」
- **镜头建议**:开场空镜 / 首帧。可做「项目从侧栏逐条淡入」的入场动画,或从这张过渡到 02。

## 02 · 项目详情(运行中)★ 主推
`devmanager-02-project-running@2x.png` — 2130×1680

- **画面**:选中 `ark-us-vue/dev`,状态「启动中」,右上局域网二维码 + `http://192.168…`,中部 `cpu / mem / uptime` 实时指标,控制按钮(停止/重启/终端/代码/目录/删除),下方彩色 Vite 日志。
- **可见功能**:实时 CPU/内存/运行时长、局域网二维码扫码调试、一键启停重启、彩色可搜索日志、端口就绪识别(`:5,173`)。
- **解说 / 字幕**:「启动、日志、指标、手机扫码调试——一屏全在这。」
- **镜头建议**:核心镜头。依次高亮四个热点(引线动画):侧栏 → 控制按钮 → 二维码 → 日志。二维码可做 zoom-in 特写。**已有标注主图 `../hero.png` 可直接用作封面帧。**

## 03 · 新建进程(弹窗)
`devmanager-03-new-process@2x.png` — 2130×1680

- **画面**:「新建进程」弹窗,目录输入(`~/my-app or browse →`)、命令 `npm run dev`、端口、「+ 添加命令」、分类 tag、添加/取消。
- **可见功能**:新增项目、多命令、端口、tag 分类、目录浏览。
- **解说 / 字幕**:「填目录和命令,10 秒加一个项目。」也可强调「支持 package.json 脚本一键导入」。
- **镜头建议**:演示「添加项目」步骤。可做字段逐个填入的打字动画。

## 04 · 设置 · MCP 接入
`devmanager-04-settings-mcp@2x.png` — 2130×1680

> ⚠️ **待替换**:此图显示的是旧包名 `@josephcc123/devmanager-mcp`。已迁移到 `@labtool/devmanager-mcp` 并重新构建,需重开 app 抓设置→MCP 页替换后再用于成片。

- **画面**:设置页 MCP 标签,给 Claude Code / Codex / Cursor 三段可复制的接入命令。
- **可见功能**:MCP 集成 —— 让 AI 工具直接列/启/停/重启/新建/删除项目、看日志、跑启动组合。
- **解说 / 字幕**:「一行命令接入,AI 直接帮你管本地进程。」
- **镜头建议**:强卖点镜头。可配「Claude 发指令 → DevManager 里项目自动启动」的联动演示(录屏 B-roll)。

## 05 · 设置 · 统计 + 启动热力图
`devmanager-05-settings-stats@2x.png` — 2130×1680

- **画面**:顶部 5 个数字(总启动次数 / 累计时长 / 最长单次 / 连续天数),GitHub 风格启动活动热力图,活动洞察 + 最常用项目。
- **可见功能**:使用统计、启动热力图、活跃时段/崩溃次数洞察。
- **解说 / 字幕**:「你的本地开发,数据看得见。」
- **镜头建议**:数字 count-up 动画 + 热力图格子逐列点亮。

## 06 · 菜单栏下拉
`devmanager-06-menubar@2x.png` — 628×442

- **画面**:菜单栏 MenuBarExtra 下拉,项目列表带端口 + 快速启动 ▶,「打开 / 退出 DevManager」。
- **可见功能**:菜单栏常驻、不开主窗也能快速启停。
- **解说 / 字幕**:「不用开窗,菜单栏点一下就跑。」
- **镜头建议**:小尺寸,适合右上角画中画 / 强调「随手可达」。

## 07 · 深色模式(项目详情)
`devmanager-07-project-dark@2x.png` — 2130×1680

> 说明:此图为「已停止」状态(无二维码/指标)。若需深色也显示「运行中 + 二维码 + CPU/内存」,切深色模式启动项目后重抓即可。

- **画面**:深色主题下的项目详情,侧栏 + 命令/路径/tag + 控制按钮 + 日志。
- **可见功能**:深色 / 浅色自动跟随系统。
- **解说 / 字幕**:「深色浅色,跟随系统自动切换。」
- **镜头建议**:与 02(浅色)做明暗切换转场,点出主题自适应。已有标注主图 `../hero-dark.png`。

1. **06 菜单栏** —— 「本地服务开开关关,很烦?」(痛点开场)
2. **01 主界面** —— 「DevManager,一个侧栏全管起来。」
